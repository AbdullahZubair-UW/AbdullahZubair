\section{Definition and Examples of Vector Spaces}

\begin{definition}{Vector space}{label}
A vector space over $\F$ is a set $V$ equipped with two operations, \emph{addition} and \emph{scalar multiplication}. The elements of $V$ are usually called \emph{vectors}\\


The addition operation takes two elements $v,w\in V$ and produces a new element $v + w\in v$, called the sum of the vectors. Addition is defined with of vectors obeys four key properties:
\begin{enumerate}
\item For all $v,w,x\in V$, $(v + w) + x = v + (w + x)$. (Associativity of addition)
\item There is a vector $0\in V$, called a \emph{zero vector}, such that $v + 0 = v$ for all $v\in V$. (Existence of a zero vector)
\item For all $v\in V$, there is a vector $-v\in V$ such that $v + (-v) = 0$. (Existence of an \emph{additive inverse}).
\item For all $v,w \in V$, $v + w = w + v$. (Commutativity of addition)

\end{enumerate}

The scalar multiplication takes an element $c \in \F$ (called the scalar) along with a vector $v\in V$, and produces $cv\in V$. Scalar multiplication of vectors obey four properties:
\begin{enumerate}
\item $\forall c_1,c_2 \in F$, $\forall v\in V$, $c_1(c_2v) = (c_1c_2)v$
\item $\forall v\in v$, $1 \cdot v = v$
\item $\forall c_1,c_2 \in F$, $\forall v\in v$, $(c_1 + c_2)v = c_1v + c_2v$
\item $\forall c\in \F$, $\forall v,w\in V$, $c(v + w) = cv + cw$
\end{enumerate}

\end{definition}

In general, a \emph{field} is a set where you can add, subtract, multiply, and divide elements from the set in "familiar" ways, as you would see in vector spaces for example.
